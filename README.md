# HOW TO SETUP
I use [Chezmoi](https://www.chezmoi.io/) to manage my dotfiles. 

```
$ brew install chezmoi
$ chezmoi init git@github.com:soya-miyoshi/dotfiles.git
$ chezmoi apply
```
